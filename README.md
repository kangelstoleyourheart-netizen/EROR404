# 🔍 MultiSearch Error404

Умный агрегатор поиска товаров и новостей с современным интерфейсом и мульти-источниками.

---

## 🚀 Возможности

* 🔎 Поиск товаров (Avito, Youla и др.)
* 📰 Агрегация новостей (Habr, VC, CNews)
* 👤 Система регистрации и входа (email-код)
* 🌙 Темная / светлая тема
* ⚡ Быстрый интерфейс (Vanilla JS)
* 🤖 Парсинг через Puppeteer
* 💾 Хранение данных (SQLite / JSON)

## 🛠️ Установка

1. Установи зависимости:

```bash
npm install
```

---

## ▶️ Запуск

### Режим разработки:

```bash
npm run dev
```

### Обычный запуск:
выполняется используя 2 команды 

```bash
1. npx http-server -p 5500

2. npm start
```

После запуска открой:

👉 http://localhost:5500
